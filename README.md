# Password generator with rust lang
Generate random passwords based on password length input.

## Source
* [`main.rs`](https://github.com/ehsan-shahbakhsh/rust-password-generator/blob/main/src/main.rs)

## Install dependencies
```shell
cargo install
```

## How to use
```shell
cargo run
```
